Teacher

Teachers


In this part user can create, update, or delete Teacher.        


Create Teacher

Follow the following steps to create Teacher

i.        Click Teacher link under the setup menu as indicated below.


 


ii.        The following screen will be displayed.


         


iii.        Enter ID Number.

iv.        Select Gender.

v.        Enter FirstName.

vi.        Enter MiddleName.

vii.        Enter LastName.

viii.        Enter UserName.

ix.        Enter Email.

x.        Enter PhoneNumber.

xi.        Select status.

xii.        Select ClassName.

xiii.        Select sectionName.

xiv.        Select Subject.

    • Click Remove link to remove assignment details to the respective teacher.
    • Click "Add New Row" button to add new assignment details to the respective teacher.



xii.        Click “Create” button to save or "Cancel" to clear all filled data.



Update Teacher

Steps

i.        Click the Teacher you want to update from the list as indicated below.


 


ii.        Edit Teacher details as required

iii.        Click Update button to save

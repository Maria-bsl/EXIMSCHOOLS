    Introduction

      1. About B-Smart academia

    B-SMART ACADEMIA is a web-based online portal designed & developed by Biz-Logic Solutions Ltd and empowered by Bank to;


    • Support Online  collection of school fees, 
    • Record and Tracking of student’s academic Performance, 
    • Plan Transportation routes and tracking pickup & drop-off of students from school buses, 
    • Record and sharing of School calendar & Time-table,
    • Manage Library activities as well as tracking movements of Library assets


      1. System requirement

    School fee management system operates in any standard browser (Preferably latest versions) through Laptop Computer, Desktop Computer, Tablets, and Smart phones. 

    The application requires network connection (be Internet or in VPN with the Server) in order to save data to database. Users do not have to install the system to their devices since it can be accessed from any location via internet or VPN.


      1. User Access Levels        

    This system has three categories of users as mentioned below.

    • Schools users
    • Administrators
    • Bank users

    NOTE: this manual is only for school users.


      1. Contingencies

    User will not be able to access the system in the absence of internet connection.
